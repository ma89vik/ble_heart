# ble_heart

# Pre-Requirements
 * CMake installed

# Building the project
 * Set TOOLCHAIN_PATH to the path to your arm-none-eabi folder if toolchain not in path
 * Set NRF_SDK_PATH to the path to your nrf52 SDK
 * Install python requirements from requirements.txt
